## Getting Help

### 1. Check Existing Resources

Before opening a new question, check:
- The [README](README.md) for common setup issues
- The [ARCHITECTURE docs](docs/ARCHITECTURE.md) for design questions
- [GitHub Issues](https://github.com/foursecondfivefour/aether-mcp-server/issues) for known bugs and feature requests

### 2. Open a Discussion

For questions, ideas, and general help:
- Start a [GitHub Discussion](https://github.com/foursecondfivefour/aether-mcp-server/discussions)

### 3. Report a Bug

If you've found a bug:
- Open a [Bug Report](https://github.com/foursecondfivefour/aether-mcp-server/issues/new?template=bug_report.md)
- Include your Windows version, AETHER_01 version, and steps to reproduce

### 4. Feature Requests

Have an idea for a new tool or enhancement?
- Open a [Feature Request](https://github.com/foursecondfivefour/aether-mcp-server/issues/new?template=feature_request.md)
